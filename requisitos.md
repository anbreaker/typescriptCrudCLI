### Paquetes requeridos

- lowDB --> Base de datos en archivo JSON (Nodejs & npm)
- inquirer.js --> Aplicaciones de Consola con Javascript (CLI Apps)
- `npm install inquirer lowdb`
- `npm install typescript @types/inquirer @types/lowdb -D`
